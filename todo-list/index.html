To-Do App

Introduction

A To-Do list Web App where the user can add tasks and check the ones that you complete. In addition you can remove the tasks completed or all of them.

  • Stack:
    • React
    • Hooks
    • Local Storage
    • PWA
Open Project
View Source

Purpose and goal

Developed to improve my skills in React, such as managing states and learn using some hooks.

Features

  • The tasks you add will be saved in localstorage. If you refresh the page, the list won't be lost.
  • It's a Progressive Web App. You can install it on a mobile and use it like a native app.

Lessons Learned

  • How to use the Localstorage to write and save data to read it later.
  • Using the spread syntax on arrays.
  • Using filter() and map() methods on arrays.
