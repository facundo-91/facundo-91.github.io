Fullstack To-Do List

Introduction

A to-do list App using a design provided from a Frontend Mentor challenge and built using React in the Front-End and ExpressJS with MongoDB in the Back-End.

  • Stack:
    • React
    • Axios
    • Express
    • MongoDB
Open Project
View Source

Purpose and goal

Developed to learn some backend by building an API server that connects with a database.

Features

  • Todo filtering by All, Active, or Completed.
  • Data fetching using Axios from an API made with Express.
  • Data persistence using MongoDB.
  • Dark Mode toggle.

Lessons Learned

  • Building a server with Express.
  • Using a NoSQL database.
  • Folder structuring of a backend project.
