Movie Browser App

Introduction

Web App for browsing movie info coming from TheMovieDB API. You can find the synopsis, duration, rating and more. Also have account creation support to bookmark your favorites.

  • Stack:
    • React
    • Context API
    • React Router
    • Firebase
    • TailwindCSS
Open Project
View Source

Purpose and goal

Developed to solidify my React knowledge, so I choose to practice by building a Full-stack app with user authentication and a database.

Features

  • User authentication with Google or with email and password using Firebase.
  • Adding your favorites movies to a list using Firestore.
  • Debounce search to reduce amount of server calls on every key stroke.

Lessons Learned

  • How to implement Firebase in the project for the authentication and the database.
  • Using the Context API to use with Firebase.
  • Routing of pages with React Router.
